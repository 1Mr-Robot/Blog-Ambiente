## Descripcion del Proyecto

Este proyecto es un blog estático desarrollado como parte de la materia de Ambiente y Sustentabilidad. Su propósito es conscientizar a los lectores sobre la importancia del cuidado del medio ambiente mediante artículos, información y recursos relacionados con la sustentabilidad, ecología y prácticas ambientales responsables.

El blog está construido utilizando tecnologías web modernas y se sirve de forma estática, lo que significa que no requiere un backend dinámico ni una base de datos. El contenido se gestiona mediante archivos Markdown, lo que facilita la edición y actualización de los artículos.

### Características principales

- Blog estático de rápido rendimiento
- Contenido gestionado mediante Markdown
- Diseño responsivo adaptable a dispositivos móviles
- Temas relacionados con el medio ambiente y sustentabilidad

### Propósito

El objetivo principal de este proyecto es servir como herramienta educativa para:
- Difundir información sobre problemas ambientales actuales
- Promover prácticas sustentables en la vida cotidiana
- Crear conciencia sobre la importancia de proteger nuestro entorno
- Fomentar la responsabilidad ambiental en la comunidad universitaria
